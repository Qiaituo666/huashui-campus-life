# 华水校园生活 - 前端项目

此目录用于存放华水校园生活服务平台的前端项目代码。

## 推荐技术栈

- **框架**: Vue 3.x + TypeScript（或 React 18.x）
- **UI 组件库**: Element Plus（Vue）/ Ant Design（React）
- **构建工具**: Vite
- **状态管理**: Pinia（Vue）/ Zustand（React）
- **HTTP 客户端**: Axios
- **地图组件**: 腾讯地图 / 高德地图 API

## 目录结构建议

```
frontend/
├── src/
│   ├── api/          # API 接口层
│   ├── assets/       # 静态资源
│   ├── components/   # 公共组件
│   ├── composables/  # 组合式函数
│   ├── layouts/      # 布局组件
│   ├── router/       # 路由配置
│   ├── stores/       # 状态管理
│   ├── styles/       # 全局样式
│   ├── utils/        # 工具函数
│   └── views/        # 页面视图
├── public/           # 公共文件
├── package.json
└── vite.config.ts
```

## 页面规划

| 模块 | 页面 | 说明 |
|------|------|------|
| 登录 | 登录/注册 | 学生/保洁/宿管统一登录 |
| 首页 | 数据看板 | 公告、水电余额概览 |
| 宿舍 | 我的宿舍 | 房间/床位/设施信息 |
| 水电 | 水电查询 | 余额、账单、用量趋势 |
| 缴费 | 在线缴费 | 微信/支付宝充值 |
| 报修 | 报修工单 | 提交/跟踪/评价 |
| 请假 | 请假申请 | 提交/查看审批状态 |
| 评价 | 宿舍评价 | 参与满意度调查 |
| 公告 | 系统公告 | 查看停水/停电/放假通知 |
| 管理 | 后台管理 | 宿管/超管功能入口 |
