## 1. 项目概述

### 1.1 项目背景

华北水利水电大学现辖 **龙子湖（主校区）、花园（老校区）、江淮（新校区）** 三个校区，在校学生规模庞大。当前宿舍管理依赖线下手工台账 + 分散的微信群通知，存在以下痛点：

| 痛点 | 现状 | 影响 |
|------|------|------|
| 水电费查询不便 | 基础电费免费，超额自缴，但学生需到宿管处排队查余额 | 体验差、效率低 |
| 报修流程冗长 | 口头报修 → 纸质登记 → 电话派单 | 响应慢、无跟踪 |
| 请假审批分散 | 纸质假条签字流转 | 易丢失、无法追溯 |
| 保洁考勤靠手写 | 签到本签名 | 代签难杜绝、统计困难 |
| 宿舍评价缺失 | 无系统性反馈渠道 | 管理优化缺少数据支撑 |
| 跨校区信息割裂 | 三校区各自为政，统一管理困难 | 超级管理员无法全局掌控 |

### 1.2 产品目标

打造一个覆盖「学生 — 保洁 — 宿管 — 超级管理员」四角色的 **三校区一体化校园宿舍生活服务平台**，核心目标：

1. **线上化**：水电查询/缴费、报修、请假审批全流程线上闭环
2. **三校区统一**：龙子湖/花园/江淮一套系统，按校区隔离数据
3. **真实数据驱动**：楼栋、房型、配置完全对齐实际校园
4. **数据化**：用水用电趋势、报修热力图、评价排行等数据分析
5. **可复制**：架构设计支持扩展到其他高校

### 1.3 产品边界

| 范围内 | 范围外（后续版本） |
|--------|---------------------|
| 宿舍水电管理 | 校园一卡通集成 |
| 报修闭环 | 图书馆座位预约 |
| 请假审批 | 课表/成绩查询 |
| 保洁考勤与任务 | 校园外卖/跑腿 |
| 宿舍评价体系 | 社团活动管理 |
| 系统公告 | 教务系统对接 |

## 2. 校园与宿舍全景

### 2.1 三校区总览

| 维度 | 龙子湖校区 | 花园校区 | 江淮校区 |

| 南苑 | 1、2、3、6、7 号楼 | 六人间 | 2 上下铺 + 2 上床下桌 | ✅ | ❌ 公共卫浴 | 2025 年翻新：全新床桌、防盗门、人脸识别门禁 |

### 2.4 江淮校区 — 楼栋详表

| 楼栋 | 房型 | 床桌 | 阳台 | 卫浴 | 特点 |
|------|------|------|------|------|------|
| 1、2、3、4 号楼（善水书院：3、4 号楼） | **四人间** | 上床下桌 | ✅ 超大阳台 | ✅ 干湿分离独立卫浴 | 华水宿舍天花板 |

> **江淮校区特点**：2021 年投用，全部四人间、无六人间；瓷砖通铺、全屋全新家具；24 小时热水不间断；楼内设书院活动室、自习室、心理驿站、一站式服务大厅；人少安静、绿化完善。

### 2.5 通用规则（三校区统一）

| 规则项 | 说明 |
|--------|------|
| 水电 | 基础电费免费，超额自缴；**严禁**电热锅、电热毯等大功率电器 |
| 床铺尺寸 | 0.9m × 2m 标准单人床 |
| 分配规则 | 按专业学院统一分配，学生**不可**自选楼栋/房型 |
| 洗浴 | 四人间独卫可随时淋浴；六人间只能使用楼层公共隔间浴室 |
| 空调 | 全部宿舍预留空调位，租赁费用学生自理 |
| 热水 | 江淮校区 24h 不间断；龙子湖/花园限时供应（各楼栋时段不同） |
