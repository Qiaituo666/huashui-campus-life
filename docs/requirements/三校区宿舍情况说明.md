# 华北水利水电大学 宿舍情况说明

> **整理时间**：2026-06-22  
> **数据来源**：实地调研 + 校内公开信息  
> **用途**：校园生活服务平台数据基线、数据库楼栋配置初始化参考  

---

## 一、学校概况

华北水利水电大学现有三个校区：

| 校区 | 位置 | 定位 | 住宿费标准 |
|------|------|------|:----------:|
| **龙子湖校区** | 郑州（主校区） | 绝大多数理工科、研究生 | 四人间 1000 元/年 |
| **花园校区** | 郑州（老校区） | 国际教育学院、乌拉尔学院、部分文科 | 六人间 800 元/年 |
| **江淮校区** | 信阳（新校区，2021年投用） | 软件工程、善水书院 | |

---

## 二、全校统一规则

以下规则三校区通用：

| 规则项 | 说明 |
|--------|------|
| 床铺尺寸 | **0.9m × 2m** 标准单人床 |
| 空调 | 所有宿舍预留空调位，**租赁费用学生自理** |
| 供暖 | 冬季统一供暖（市政集中供暖） |
| 直饮水 | 每层楼配有直饮水机 |
| 洗衣机 | 每层楼配有扫码支付洗衣机 |
| 分配规则 | 按专业学院统一分配，学生**不可**自选楼栋/房型 |
| 水电 | 基础电费免费，**超额自缴**；严禁电热锅、电热毯等大功率电器 |
| 洗浴方式 | 四人间独卫可随时淋浴；六人间使用楼层公共隔间浴室（带门） |

## 八、对系统设计的影响

基于以上真实宿舍数据，系统需要支持：

1. **校区三级结构**：校区 → 片区（一区/二区/南苑/北苑）→ 楼栋 → 房间 → 床位
2. **房型差异化展示**：四人间独卫随时洗浴 vs 六人间公浴隔间，前端须区分布局
3. **配置继承**：同一片区楼栋配置高度一致，支持楼栋配置模板
4. **热水时段管理**：江淮 24h 不间断，龙子湖/花园限时供应，需楼栋级配置
5. **床桌布局差异**：龙子湖统一上床下桌，花园有混合布局，江淮统一，数据不可一刀切
6. **空调规则**：全部预留空调位但需租赁，系统需区分「已租」和「可租」状态
7. **分配不可自选**：按学院自动分配，前端不提供选房功能，只展示分配结果
