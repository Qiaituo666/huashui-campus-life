## 项目框架一览

代码

```
E:\Workspace\demos\water_compus\huashui-campus-life\
│
├── pom.xml                          ← 根 POM（Spring Boot 3.5.3 + Cloud 2025.0.0 + Alibaba 2025.0.0.0）
│
├── huashui-common/                  ← 公共模块（无 Application，AutoConfiguration 自动装配）
│   ├── config/   RedisConfig, JacksonConfig, SaTokenConfig
│   ├── exception/  BusinessException, GlobalExceptionHandler
│   ├── response/   Result<T>, PageResult<T>
│   ├── constants/  RedisConstants, MQConstants
│   └── annotation/ @OperationLog
│
├── huashui-gateway/      :8080     ← Gateway + AuthFilter（白名单 + Sa-Token 登录校验）
├── huashui-auth/         :8081     ← AuthController（login/logout/captcha 骨架）
├── huashui-user/         :8082     ← 用户模块（学生/员工 CRUD 预留）
├── huashui-dormitory/    :8083     ← 核心宿舍模块（含 Feign 调用 + RabbitMQ）
├── huashui-utility/      :8084     ← 水电模块（含 @Scheduled 定时同步 Job）
├── huashui-attendance/   :8085     ← 考勤模块（Feign 调 dormitory 查请假）
├── huashui-message/      :8086     ← 消息中心（RepairConsumer + LeaveConsumer + PayConsumer）
├── huashui-system/       :8087     ← 系统模块（@OperationLog AOP 切面）
├── huashui-file/         :8088     ← 文件模块（MinIO 集成预留）
└── huashui-client/                 ← Feign 接口集中（4 个 API 包，被其他模块依赖）
```
